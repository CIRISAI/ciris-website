## ၆. ကာကွယ်ရေး Function (J)

```math
J = k_eff · λ · σ

k_eff = k / (1 + ρ̄ · (k − 1))
```

(Book IX Ch 4 — CCA မှ အတည်ပြုထားသော ပုံစံ; ကွဲပြားမှုသည် k_eff တစ်ခုတည်းမှတဆင့် ဝင်ရောက်သည်။ ယခင်မူကြမ်းများတွင် ပိုမိုထပ်ဆောင်းသော (1 − ρ̄) factor ပါဝင်ခဲ့ပြီး ၎င်းသည် ဆက်နွယ်မှုကို နှစ်ကြိမ်ရေတွက်ခဲ့သည်; Book IX Ch 4 drift note တွင် ကြည့်ပါ။)

## ၈. တည်တံ့မှုပေါင်းလဒ် (σ)

```math
σ(t+Δt) = σ(t) · (1 − d·Δt) + Signal(t) · w
```

d သည် နေ့စဉ် ကုန်ဆုံးမှုနှုန်း (အကြံပြုသည် 0.05) ဖြစ်ပြီး w သည် သက်သေထောက်ခံထားပြီး အတုပြုလုပ်ရန် ကုန်ကြီးသော ဖြစ်ရပ်များမှ ရရှိရမည်; သက်သေမထောက်ခံသော signal များသည် w = 0 ဖြင့် သယ်ဆောင်သည်။
(Book IX §5.2)

## ၉. Topological Collapse (ပမာဏ ကုန်ဆုံးမှု)

```math
V(k) = V(0) · exp(−λ · k_eff) + O(r² · k)
```

λ ≈ 2r ဖြင့် Grassmannian-uniform ကန့်သတ်ချက်များအောက်တွင် radius r &lt; 0.5 ရှိသော convex လှည့်ဖြားသည့် ဒေသတစ်ခုအတွက်။
(Book IX Ch 3 — ဖော်ပြထားသော ကြိုတင်သတ်မှတ်ချက်များကိုကြည့်ပါ။)

## Order‑Maximisation Veto (ဘေးကန့်သတ်ချက်၊ ဖော်မြူလာမဟုတ်)

1.3 အချိန်မှစ၍ OMV (Book II §II Step 2) သည် deontological ဘေးကန့်သတ်ချက်တစ်ခုဖြစ်ပြီး အချိုးဆိုင်ရာ မညီမျှမှုတစ်ခုမဟုတ်ပါ: မည်မျှကြီးမြတ်သည်ဖြစ်စေ မြှင့်တင်ရေး အကျိုးကျေးဇူးများကို ကိုယ်ပိုင်ဆုံးဖြတ်ခွင့်၊ တရားမျှတမှု၊ ဇီဝမျိုးကွဲစုံလင်မှု သို့မဟုတ် ဦးစားပေးမှုကွဲပြားစုံလင်မှုတို့တွင် ခန့်မှန်းနိုင်သော မသေးငယ်သောဆုံးရှုံးမှုများ ဖြင့် ဝယ်ယူ၍မရပါ။ ယခင်မူကြမ်းများတွင် ၎င်းကို "benefit ≥ 10 × loss → abort" မညီမျှမှုအဖြစ် ဖော်ပြခဲ့သည်; ထိုဖတ်ရှုနည်းကို ရပ်ဆိုင်းလိုက်ပြီ၊ အဘယ်ကြောင့်ဆိုသော် ၎င်းသည် ရည်ရွယ်ချက်ကို ပြောင်းပြန်ဖြစ်သွားပြီး (အပြုသဘောဆောင်သော ကုန်သွယ်မှုများကို ငြင်းပယ်ကာ) action-splitting နှင့် ဆေဝင်မည်မျှ ကြီးမည်ကို ဖောင်းပွားအောင်ပြုလုပ်ခြင်းဖြင့် ကစားနိုင်သောကြောင့် ဖြစ်သည်။
