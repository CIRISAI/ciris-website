## 6. Defense Function (J)

```math
J = k_eff · λ · σ

k_eff = k / (1 + ρ̄ · (k − 1))
```

(Book IX Ch 4 — CCA-சரிபார்க்கப்பட்ட வடிவம்; பன்முகத்தன்மை k_eff மட்டுமே மூலம் நுழைகிறது. முந்தைய வரைவுகளில் கூடுதலாக ஒரு (1 − ρ̄) காரணி இருந்தது, அது தொடர்பை இரட்டிப்பாகக் கணக்கிட்டது; Book IX Ch 4 திரிவு குறிப்பைப் பார்க்கவும்.)

## 7. Flourishing Capacity (F)

```math
F = k_eff · λ · σ
```

(Book IX Ch 6 — J உடன் உறுப்பு உறுப்பாக சரியாக ஒத்திருக்கிறது; பாதுகாப்பு என்றில்லாமல் ஆக்கத்திறன் கொண்டதாக விளக்கப்படுகிறது. முன்பு C என எழுதப்பட்டது; CIRIS Capacity Score 𝒞_CIRIS இன் மைய-அடையாள காரணி C உடன் மோதலைத் தவிர்க்க 1.3-RC2 இல் மறுபெயரிடப்பட்டது — உறவின் அதிகார அறிவிப்பாக விளங்கும் Book IX Ch 6 பெயரியல் குறிப்பைப் பார்க்கவும்.)

## 8. Sustainability Integral (σ)

```math
σ(t+Δt) = σ(t) · (1 − d·Δt) + Signal(t) · w
```

இங்கு d என்பது நாளாந்த சிதைவு வீதம் (பரிந்துரைக்கப்பட்டது 0.05), மேலும் w ஆனது சான்றளிக்கப்பட்ட, போலியாக்குவதற்கு அதிக செலவு தேவைப்படும் நிகழ்வுகளிலிருந்து பெறப்பட்டிருக்க வேண்டும்; சான்றளிக்கப்படாத சமிக்ஞைகளுக்கு w = 0 ஆகும்.
(Book IX §5.2)

## 9. Topological Collapse (volume decay)

```math
V(k) = V(0) · exp(−λ · k_eff) + O(r² · k)
```

Grassmannian-சீரான தடைகளின் கீழ் r &lt; 0.5 ஆரமுள்ள குவிந்த வஞ்சக மண்டலத்திற்கு, λ ≈ 2r.
(Book IX Ch 3 — கூறப்பட்ட முன்நிபந்தனைகளைப் பார்க்கவும்.)

## Order‑Maximisation Veto (side-constraint, not a formula)

1.3 நிலையில், OMV (Book II §II Step 2) என்பது ஒரு கடமையியல் பக்கத்தடையாகும் (deontological side-constraint), விகித சமன்பாடல்ல: எவ்வளவு பெரிதாக இருந்தாலும் உகப்பாக்கல் நன்மைகளை, தன்னாட்சி, நீதி, உயிரினப் பன்முகத்தன்மை அல்லது விருப்ப பன்முகத்தன்மையில் அற்பமல்லாத முன்னறிவிக்கப்பட்ட இழப்புகள் மூலம் வாங்கிக்கொள்ளலாகாது. முந்தைய பதிப்புகள் இதை "benefit ≥ 10 × loss → abort" சமன்பாடாக வெளிப்படுத்தியது; அந்த வாசிப்பு நிறுத்தப்பட்டது, ஏனெனில் அது நோக்கத்தை தலைகீழாக்கியது (சாதகமான பரிவர்த்தனைகளை தடுத்தது) மேலும் செயல்-பிளவு மற்றும் வகுத்தல் பெரிதாக்கல் மூலம் சூதாடத்தக்கதாக இருந்தது.
