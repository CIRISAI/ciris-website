## எதிரி மதிப்பாய்வுக்கான அழைப்பு

CIRIS 1.3-RC2-ஐ அழுத்தி சோதிக்க பாதுகாப்பு ஆய்வகங்கள், சுதந்திர ஆராய்ச்சியாளர்கள், மற்றும் சிவில் சமூக அமைப்புகளை நாங்கள் அழைக்கிறோம்.

சிக்கல்களை **https://github.com/CIRISAI/CIRISAccord/issues** இல் சமர்ப்பிக்கவும்.

முன்னுரிமை தலைப்புகள்:
* metric-Goodhart காட்சிகள் (σ-சான்றளிப்பு சுற்றுவழியை உள்ளடக்கியது — Book IX §5.2 காண்க)
* board-capture பாதைகள் மற்றும் Book VIII Ch 9 நிர்வாக ஏற்பாடு
* Annex F சம்பவ செயல்முறைகள் முழுவதும் நிலை உயர்த்தல் தோல்விகள்
* §9.2-இல் பெயரிடப்பட்ட Book IX அனுமானங்கள் (சத்திய-உள்ளடக்கம், LLM தொடர்பு தரை, தகவமைப்பற்ற எதிரி)
* RC தேவை 4-இன் கீழ் செய்யப்பட வேண்டிய red-team பயிற்சி (அனைத்து CIRIS சோதனை புள்ளிகளிலும் தேர்ச்சி பெற முயற்சிக்கும் ஒரு கற்பனையான தவறான சீரமைப்பு உகப்பாக்கி)

சரிபார்க்கப்பட்ட முக்கியமான கண்டுபிடிப்புகளுக்கு பரிசுகள் கிடைக்கும் (Bug-Bounty Ledger, Book I Ch 2).

பதிப்பு சுருக்கம்:

* **2025-04-16 — 1.0-β** ஆரம்ப வெளியீடு; அபாய-வரையறுக்கப்பட்டது; 24-மாத மறைவு.
* **1.2-Beta** — முழு Book IX வடிவியல் கட்டமைப்பை கொண்டுசெல்லும் செயல்-வரைவு வரி; மூன்று சறுக்கிய சித்தரிப்புகளிலிருந்து நியமனஞ்சார் களஞ்சியத்தில் ஒருங்கிணைக்கப்பட்டது (import commit `000ba57`).
