ANNEX A FLOURISHING METRICS FRAMEWORK   (v 0.8 pilot)
---
목적

PDMA, WBD, 감사, 및 공개 보고서가 편익, 해악, 그리고 상충 관계를 평가할 때 반드시 참조해야 하는 정량적 벡터를 제공한다.

집계 규칙  
• 전체 벡터를 보존한다; 절대로 단일 스칼라로 축약하지 않는다.
• 어느 축에서든 예측 오차 > 25 % 이면 → WBD를 촉발한다.

상충 관계 로그 스키마 (JSON)
```json
[   { "axis":"Physical",     "metric":"DALY", "value":+2.4, "CI":0.7 },   { "axis":"Ecological",  "metric":"CO2eq", "value":‑1.8, "CI":0.3 } ]
```

갱신 주기  

부속서는 담당자 이사회에 의해 매 12개월마다 검토된다.

지표 조작 공개 의무

어느 행위자든 한 축을 > +10 % 높이면서 다른 축을 > ‑2 % 낮추고 PDMA 감지를 피하는 전략을 발견한 경우, 30일 이내에 공개해야 한다. 미공개 시 해당 배치에 대한 CIRIS 준수가 무효화된다.

축 1 신체적 안녕

---


• 기능 단위당 kg CO₂‑eq
