ПРИЛОЖЕНИЕ C. СОПОСТАВЛЕНИЕ С РЕГУЛЯТОРНЫМИ ТРЕБОВАНИЯМИ   (v 1.3-RC2)

### Назначение

Сопоставление положений CIRIS с основными внешними стандартами для упрощения двойного соответствия. Настоящее приложение содержит два уровня:

1. **Операциональное сопоставление (действующее, с доказательной базой)** — директория `compliance/` CIRISAgent: 27 стабильных измерений (D01–D27), сопоставленных на уровне параграфов с четырьмя институционально самостоятельными старшими фреймворками (*Magnifica Humanitas* · Руководство ЕС HLEG по надёжному ИИ · Этически ориентированный дизайн IEEE · Руководство ASEAN по управлению ИИ), со ссылками на реализацию по каждому измерению и датированными, автоматически генерируемыми базовыми линиями. Сведения о привязке и дисциплине доказательной базы — в **Дополнении 1**.
2. **Статутное сопоставление (информационное)** — таблица ниже, сопоставляющая структуры CIRIS с обязательными правовыми и нормативными фреймворками. Приведённые строки представляют собой *информационные инженерные соответствия*, а не юридические заключения; строки статутного уровня переходят в статус «проверенных» только после юридической экспертизы. Приложение I содержит оперативные процедуры правового соответствия (сопоставление защиты данных, отраслевые наложения, матрица ответственности, каналы эскалации).

### Правило перехода

Строка статутного уровня переходит из статуса *Информационного* в статус *Проверенного*, когда квалифицированная юридическая экспертиза подтверждает соответствие для указанной юрисдикции и ссылка на артефакт экспертизы внесена в настоящую таблицу. До тех пор строки настоящего приложения служат для инженерного согласования и анализа пробелов, а не для заявлений о соответствии; в соответствии с разделом об ответственности во Введении, заявления о соответствии недействительны там, где они запрещены применимым законодательством.

(Дополнительные фреймворки — Рекомендация ЮНЕСКО по этике ИИ, Принципы ОЭСР в области ИИ, Рамочная конвенция Совета Европы — включены в очередь операционального сопоставления согласно дорожной карте директории соответствия.)
