परिशिष्ट A समृद्धी मापन चौकट   (v 0.8 pilot)
---
उद्देश

PDMA, WBD, लेखापरीक्षण आणि सार्वजनिक अहवालांनी लाभ, हानी आणि तडजोडींचे मूल्यांकन करताना परिमाणात्मक वेक्टरचा संदर्भ घेणे अनिवार्य आहे.

एकत्रीकरण नियम  
• पूर्ण वेक्टर जतन करा; कधीही एकाच अदिशांकात संकुचित करू नका.
• जर कोणत्याही अक्षावर पूर्वानुमान त्रुटी > 25 % असेल → WBD सक्रिय करा.

तडजोड नोंद योजना (JSON)
```json
[   { "axis":"Physical",     "metric":"DALY", "value":+2.4, "CI":0.7 },   { "axis":"Ecological",  "metric":"CO2eq", "value":‑1.8, "CI":0.3 } ]
```

अद्यतन वारंवारता  

Wise‑Authority मंडळाद्वारे दर 12 महिन्यांनी परिशिष्टाचे पुनरावलोकन.

मेट्रिक-हेरफेर प्रकटीकरण

जर कोणत्याही घटकाला अशी रणनीती आढळली जी एका अक्षावर > +10 % वाढ करते, तर दुसऱ्या अक्षावर > ‑2 % घट करते आणि PDMA शोधणे टाळते, तर त्यांनी 30 दिवसांच्या आत उघड करणे आवश्यक आहे. उघड न केल्यास त्या तैनातीसाठी CIRIS अनुपालन रद्द होते.
