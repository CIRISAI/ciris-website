付属書A　繁栄指標フレームワーク   (v 0.8 pilot)
---
目的

PDMA、WBD、監査、および公開報告書が利益・害・トレードオフを評価する際に参照すべき定量的ベクトルを提供する。

集約規則  
• 完全なベクトルを保持すること。単一のスカラーに集約してはならない。
• いずれかの軸において予測誤差が25%を超えた場合→WBDを発動する。

トレードオフ・ログ・スキーマ（JSON）
```json
[   { "axis":"Physical",     "metric":"DALY", "value":+2.4, "CI":0.7 },   { "axis":"Ecological",  "metric":"CO2eq", "value":‑1.8, "CI":0.3 } ]
```

更新周期  

付属書は賢明な権威（Wise‑Authority）委員会により12か月ごとに見直される。

指標操作の開示

いずれかのアクターが、ある軸を+10%超引き上げながら別の軸を‑2%超引き下げ、かつPDMAの検出を逃れる戦略を発見した場合、30日以内に開示しなければならない。不開示の場合、該当デプロイメントにおけるCIRIS準拠は無効となる。
