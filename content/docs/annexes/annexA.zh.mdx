附录A　繁荣指标框架　（v 0.8 试行）
---
目的

提供量化向量，PDMA、WBD、审计及公开报告在评估收益、伤害与权衡时必须参照此框架。

聚合规则  
• 保留完整向量；切勿折叠为单一标量。
• 若任意轴的预测误差 > 25 %，则触发WBD。

权衡日志模式（JSON）
```json
[   { "axis":"Physical",     "metric":"DALY", "value":+2.4, "CI":0.7 },   { "axis":"Ecological",  "metric":"CO2eq", "value":‑1.8, "CI":0.3 } ]
```

更新周期

本附录每12个月由智慧权威委员会审查一次。

指标博弈披露

若任何行为者发现某策略能使某一轴提升 > +10 %，同时使另一轴下降 > ‑2 % 且规避PDMA检测，则必须在30天内予以披露。未披露者将丧失该部署的CIRIS合规资格。
